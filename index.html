Usage

This is default usage.

Javascript

HTML Structure


	<div id="picker"></div>

Options

Create with options


	$('#picker').felixSlider({
		colors : DefaultColorArray	// Colors
		pickerMode : false,		// ture or false : if it's true, ColorPicker show up after click event. if it's false, ColorPicker append into element.
		pickerWidth : 11,		// each color's width
		pickerHeight : 11,		// each color's height
		layerWidth : 204,		// ColorPicker layer's width
		layerHeight : 85,		// ColorPicker lyaer's height
		layerStyle : 'border: 1px solid black; padding: 2px; background-color: white;',
						// ColorPicker's style setup
		pickerStyle : 'border: 1px solid #A0A0A0; margin: 2px',
						// each color's style setup
		select : null,			// color select event
		zIndex : 10000			// z-index value
	});

Option change of created slider

	
	// Getter
		$('#picker').felixSlider('option', 'colors');
	// Setter
		$('#picker').felixSlider('option', 'colors', ['#FFCCFF', '#FF99FF', '#CC66CC', '#66FFFF', '#FF9933', '#000066']);

	// Getter
		$('#picker').felixSlider('option', 'pickerMode');
	// Setter
		$('#picker').felixSlider('option', 'pickerMode', true);

	// Getter
		$('#picker').felixSlider('option', 'pickerWidth');
	// Setter
		$('#picker').felixSlider('option', 'pickerWidth', 10);

	// Getter
		$('#picker').felixSlider('option', 'pickerHeight');
	// Setter
		$('#picker').felixSlider('option', 'pickerHeight', 10);

	// Getter
		$('#picker').felixSlider('option', 'layerWidth');
	// Setter
		$('#picker').felixSlider('option', 'layerWidth', 200);

	// Getter
		$('#picker').felixSlider('option', 'layerHeight');
	// Setter
		$('#picker').felixSlider('option', 'layerHeight', 90);

	// Getter
		$('#picker').felixSlider('option', 'layerStyle');
	// Setter
		$('#picker').felixSlider('option', 'layerStyle', 'border: 1px solid black; padding: 2px; background-color: white;');

	// Getter
		$('#picker').felixSlider('option', 'pickerStyle');
	// Setter
		$('#picker').felixSlider('option', 'pickerStyle', 'border: 1px solid #A0A0A0; margin: 2px');

	// Getter
		$('#picker').felixSlider('option', 'select');
	// Setter
		$('#picker').felixSlider('option', 'select', function (color){
			alert(color);
		});

	// Getter
		$('#picker').felixSlider('option', 'zIndex');
	// Setter
		$('#picker').felixSlider('option', 'zIndex', 1000);

Methods

You can call methods of created slider.


	// Destroy method
	$('#picker').felixSlider('destroy');
